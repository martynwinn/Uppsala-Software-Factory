4 INTRODUCTION

This program can be used to generated an SSE file for DEJAVU from a PDB file of a protein (SSE = Secondary Structure Element(s)). GETSSE replaces the old "make_sse" script and the programs PRO1 and PRO2. Operation should be trivial: 

      
 ----- EXAMPLE ----- EXAMPLE ----- EXAMPLE ----- EXAMPLE ----- EXAMPLE -----
 % 575 gerard sarek 21:22:46 progs/secs > GETSSE
